# iBrawl - V1

iBrawl is a Brawl Stars mod currently in development, all young and without any grandiose additions. For now, just a few adjustments:

- Tick : Health, Damage
- Max : Speed, Health, Damage
- 8-Bit : Speed
- Colt : Damage, Bullet
